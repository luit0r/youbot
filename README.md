Youbot is a Shell Script to increase views on Youtube using Tor ips.

### Usage:
```
git clone https://github.com/thelinuxchoice/youbot
cd youbot
service tor start
sudo ./youbot.sh URL
```

### Install requirements (xdotool, proxychains, tor):

```
./install.sh
```
